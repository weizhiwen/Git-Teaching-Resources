# 学用Markdown

## 1.1 标题

\# 开头，几级标题，几个 # 号。



## 1.2 文本

**倾斜** ：成对的单 \* 号或者单 _ (下划线)

**加粗** ：成对的双 \* 号或者双 __ (下划线)

**倾斜并加粗** ：成对的三 \* 号或者三 ___ (下划线)

**删除线** ：成对的双 ~~（波浪线）

**分割线** ：三 ---（短杆）****

**引用** ：单 > 号

**行内代码** ：成对的单 `（反引号）

**块内代码** ：成对的三 ```（反引号）

**列表** ：单 - （短杆）和数字

**链接** ：\[链接文字\]\(url地址\)
